# محصولات (Products)

این فایل JSON شامل تمام محصولات فروشگاه است.

## ساختار فایل

هر محصول باید دارای فیلدهای زیر باشد:

- `id`: شماره یکتای محصول (عدد)
- `name`: نام محصول (متن فارسی)
- `description`: توضیحات محصول (متن فارسی)
- `price`: قیمت محصول (متن فارسی با فرمت: "X,XXX,XXX تومان")
- `weight`: وزن محصول (متن فارسی)
- `grade`: درجه کیفیت محصول (متن فارسی)
- `image`: مسیر تصویر محصول (مثال: "/img/products/product-name.jpg")

## نحوه افزودن محصول جدید

1. یک ردیف جدید به آرایه JSON اضافه کنید
2. `id` را به صورت یکتا تنظیم کنید (باید با سایر محصولات متفاوت باشد)
3. تصویر محصول را در پوشه `public/img/products/` قرار دهید
4. مسیر تصویر را در فیلد `image` وارد کنید

## مثال

```json
{
  "id": 7,
  "name": "زعفران ویژه",
  "description": "توضیحات محصول...",
  "price": "۳,۵۰۰,۰۰۰ تومان",
  "weight": "۱.۵ گرم",
  "grade": "ممتاز",
  "image": "/img/products/saffron-special.jpg"
}
```

## نکات مهم

- فایل JSON باید معتبر باشد (از ویرایشگر JSON استفاده کنید)
- مسیر تصاویر باید با `/img/products/` شروع شود
- تصاویر باید در پوشه `public/img/products/` قرار گیرند
- فرمت تصاویر می‌تواند JPG، PNG، یا WebP باشد
