# چیلو - فروشگاه زعفران ممتاز

یک وب‌سایت استاتیک Next.js برای فروشگاه زعفران با طراحی فارسی و راست‌به‌چپ.

## ویژگی‌ها

- ✅ طراحی فارسی و راست‌به‌چپ (RTL)
- ✅ صفحه اصلی، درباره ما، و محصولات
- ✅ سبد خرید با مدیریت وضعیت
- ✅ فرم سفارش با ارسال ایمیل
- ✅ **ادغام با WordPress به عنوان بک‌اند**
- ✅ پشتیبانی از WooCommerce
- ✅ بهینه‌سازی شده برای SEO
- ✅ طراحی واکنش‌گرا (Responsive)
- ✅ رنگ‌بندی قرمز، بنفش و زرد

## نصب و راه‌اندازی

### 1. نصب وابستگی‌ها

```bash
npm install
```

### 2. راه‌اندازی WordPress (برای مدیریت محصولات)

برای اتصال به WordPress به عنوان بک‌اند، لطفاً فایل `SETUP_WORDPRESS.md` را مطالعه کنید.

**نکته:** اگر WordPress تنظیم نشود، فروشگاه به صورت خودکار از فایل `data/products.json` استفاده می‌کند.

### 3. راه‌اندازی EmailJS (برای ارسال ایمیل سفارشات)

برای فعال‌سازی ارسال ایمیل سفارشات، لطفاً فایل `SETUP_EMAILJS.md` را مطالعه کنید.

### 4. اجرای پروژه

```bash
npm run dev
```

سایت در آدرس `http://localhost:3000` در دسترس خواهد بود.

### 5. ساخت نسخه استاتیک

```bash
npm run build
```

فایل‌های استاتیک در پوشه `out` ایجاد می‌شوند.

## ساختار پروژه

```
chiiloo/
├── app/                 # صفحات Next.js
│   ├── page.tsx         # صفحه اصلی
│   ├── about/           # صفحه درباره ما
│   ├── products/        # صفحه محصولات
│   └── cart/            # صفحه سبد خرید
├── components/          # کامپوننت‌های قابل استفاده مجدد
├── contexts/            # Context API برای مدیریت وضعیت
└── public/              # فایل‌های استاتیک
```

## تکنولوژی‌ها

- Next.js 14
- React 18
- TypeScript
- CSS Modules
- WordPress REST API
- WooCommerce REST API (اختیاری)
- EmailJS (برای ارسال ایمیل)

## پشتیبانی

- برای راه‌اندازی WordPress، فایل `SETUP_WORDPRESS.md` را مطالعه کنید.
- برای راه‌اندازی EmailJS، فایل `SETUP_EMAILJS.md` را مطالعه کنید.
